# SubstrateMC

SubstrateMC is a blazingly fast and modern minecraft managment server it allows people to build Minecraft panels
without worrying about the minecraft process and the underlying difficulties.

## Features

- Streaming console output over WebSocket
- Starting servers
- Stopping servers
- Creating servers
- Deleting Servers
- Lightweight backend in Rust

## Goals 

- Support anything real Minecraft panels support
- Be a fast alternative to other panels
- Make it easy to create complex server panels

## None Goals

- Provide a graphical interface
- Support Bedrock servers
- Support versions < 1.20
